MarkDown绘制UML时序图

时序图

时序图Sequence Diagram描述了消息在生命线上按照约定顺序执行的一种交互行为。用图形来表示某个系统的时许关系。

通过描述对象之间发送消息的时间顺序来显示多个对象之间的交互关系。

  • 实心箭头表示同步消息 非实心箭头表示异步消息
  • 实线表示主动发送消息，虚线表示返回（应答）消息

其包含四类元素:

  • 对象 object
  • 生命线 lifeline
  • 消息 message
  • 激活 activation
