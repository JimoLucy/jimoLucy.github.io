JAVA位运算整理

位操作概述

java中位操作有 与(&)、或(|)、异或(^)、非(~)、左移(<<)、算术右移(>>)、逻辑右移(>>>)操作。

  • 按位与操作: 对应位均为1，则结果为1.否则为0
  • 按位或操作: 对应为均为0，则结果为0.否则为1
  • 异或操作: 又称不进位加法操作，是指对应位是不是不一样,对应位相同为0，不同为1
  • 按位非操作: 一元操作符，每一位取反。
  • 左移操作符(A<<B): 将A的二进制整体向左移B位，不足位补0
  • 算术右移(A>>B):将A的二进制每一位向右移B位，左边不足的补符号位的数
  • 逻辑右移(A>>>B): 将A的二进制每一位向右移B位，左边不足的补0
二进制原码、反码、补码

二进制数在内存中以补码形式存储。

  • 反码: 正数的反码与原码相同。负数的反码为原码除符号位外其余为取反
  • 补码: 正数的补码与原码相同。负数的补码为其反码+1

补码转源码:
高位为0 为正数，源码即为补码

高位为1为负数，源码为负数的补码+1

异或操作

数值两两相同为否，数值不同为真。 c/java语言中写作A^B,也被称为不进位加法

性质:

  • 交换律: A^B = B^A
  • 结合律：A^(B^C) = (A^B)^C
  • 恒等律: A^0 = A
  • 归零律: A^A = 0
  • 自反律：A^B^B = A^0 = A

位操作符总结

  • 与0(&)结果为0 与1(&)结果为原值
  • 与0(|)结果为原值 与1(|)结果为1
  • 与0(^)结果为原值 与1(^)是相反值
  • 0与其他值(| ^)均为原值，与其他值&均为0
  • 1与其他值(&)均为原值，与其他值(|)均为1,与其他值(^)为相反值
  • a^b结果中为0的说明该位二者无差别，为1的说明该位有差别
  • 相等值&为原值，相等值|为原值 相等值^为0
  • 不等值&为0，不等值|为1，不等值^为1

特殊的二进制数

  • 1 = 0b 0000 0000 0000 0000 0000 0000 0000 0001
  • -1 = 0b 1111 1111 1111 1111 1111 1111 1111 1111
  • 16 = 0b 0000 0000 0000 0000 0000 0000 0001 0000
  • 15 = 0xF = 0b 0000 0000 0000 0000 0000 0000 0000 1111
  • 256 = 1<<8 = 0b 0000 0000 0000 0000 0000 0001 0000 0000
  • 255 = 0xFF = (1 << 8) - 1 = 0b 0000 0000 0000 0000 0000 0000 1111 1111
  • ~(1<<8) == 0b 1111 1111 1111 1111 1111 1110 1111 1111

特殊技巧

  • a^b结果中为0的说明该位二者无差别，为1的说明该位有差别
  • x & (x - 1)用于将x的最后一位1变为0: x&(x-1) == 0 说明x中只有一个1，那么x为2的幂
  • x &(-x): 最后一位1的位置
  • 判断奇偶:二进制最后一位为1为奇数，为0则为偶数。x&1==0偶数
  • 变换符号: 取反+1: ~a + 1
  • 求绝对值: a >> 31 == -1 则a为负数 |a| = ~a+1 ,a >> 31 == 0为正数 |a| = a
  • 将 x 的第 n 位置1，可以通过 x |= (1 << n) 来实现
  • 将 x 的第 n 位清0，可以通过 x &= ~(1 << n) 来实现
  • 取出 x 的第 n 位的值，可以通过 (x >> n) & 1 来实现
  • 交换a，b的值：a = a ^ b; b = a ^ b; a = a ^ b;
  • 快速比较a,b是否相等： a ^ b == 0
  • 特定位翻转: 翻转10100001第六位: 10100001 ^ 00100000
  • 判断二进制数中1的数量是奇数还是偶数: 10100001: 1^0^1^0^0^0^0^1 结果为1 表示奇数个1 ，为0就是偶数个1
  • 校验和恢复主要利用的了异或的特性：IF a ^ b = c THEN a ^ c = b 应用：一个很好的应用实例是RAID5，使用3块磁盘（A、B、C）组成RAID5阵列，当用户写数据时，将数据分成两部分，分别写到磁盘A和磁盘B，A ^ B的结果写到磁盘C；当读取A的数据时，通过B ^ C可以对A的数据做校验，当A盘出错时，通过B ^ C也可以恢复A盘的数据。
  • 一个整型数组里除了N个数字之外，其他的数字都出现了两次，找出这N个数字:
    N 为1时:  [a,b,c,d,c,d,b] : a^b^c^d^c^d^b == a
N 为2时:  c = a^b 
                  通过d = c & ~(c-1) 找到c中第一个为1的的数，说明a,b在此位的值不同。根据这个特点，找出所有这个位数上为1的数(x & d > 0) 进行异或，得到的就是a或者b,然后列外的值为 c ^ a

功能 示例 位运算
去掉最后一位 (101101->10110) x >> 1
在最后加一个0 (101101->1011010) x << 1

把最后一位变成0 (101101->101100) x & ~1

把右数第k位变成1 (101001->101101,k=3) x (1 << (k-1))
把右数第k位变成0 (101101->101001,k=3) x & ~(1 <<(k-1))
右数第k位取反 (101001->101101,k=3) x ^ (1 << (k-1))
取末三位 (1101101->101) x & (1<<k - 1)
取末k位 (1101101->1101,k=5) x & (1<<k - 1)
取右数第k位 (1101101->1,k=4) x>>(k-1) & 1
把末k位变成1 (101001->101111,k=4) x ((1<<k) - 1)
末k位取反 (101001->100110,k=4) x ^ ((1 << k) - 1)
把右边连续的1变成0 (100101111->100100000)
把右起第一个0变成1 (100101111->100111111)
把右边连续的0变成1 (11011000->11011111)
取右边连续的1 (100101111->1111) ( x ^ (x+1)) >> 1
去掉右起第一个1的左边 (100101000->1000) x & (x ^ (x - 1))
